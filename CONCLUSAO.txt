┌────────────────────────────────────────────────────────────────────────────┐
│                      🚀 COMEÇAR O PROJETO                                 │
└────────────────────────────────────────────────────────────────────────────┘

    1️⃣  Crie o banco de dados:
        createdb revenue_db

    2️⃣  Execute os scripts SQL:
        psql -U postgres -d revenue_db -f scripts/001-create-tables.sql
        psql -U postgres -d revenue_db -f scripts/002-seed-admin.sql

    3️⃣  Configure as variáveis:
        Revenue-node/.env
        Revenue-app/.env.local

    4️⃣  Instale dependências:
        cd Revenue-node && npm install
        cd ../Revenue-app && npm install

    5️⃣  Inicie os servidores:
        Terminal 1: cd Revenue-node && npm start
        Terminal 2: cd Revenue-app && npm run dev

    6️⃣  Acesse a aplicação:
        🔗 http://localhost:3000
