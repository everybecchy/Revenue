                        🚀 PRÓXIMOS 5 MINUTOS


    1. Leia o README.md
       → Entenda o que foi criado

    2. Execute QUICK_START.sh
       → Setup automático (ou siga manualmente)

    3. Configure as variáveis de ambiente:
       → Revenue-node/.env
       → Revenue-app/.env.local

    4. Inicie os servidores:
       → Terminal 1: cd Revenue-node && npm start
       → Terminal 2: cd Revenue-app && npm run dev

    5. Acesse em http://localhost:3000
       → Login com admin@bravopanel.com / Mudar123
